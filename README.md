# Workshop AI Assistant

Voice-activated AI assistant for workshop environment.

## Current Status
~~Phase 0 - Foundations~~
Phase 1 - Voice Pipeline

## Prerequisites
- Raspberry Pi 5 (8GB)
- Python 3.11+
- USB audio device

## Setup

### System Dependencies
```bash
sudo apt update
sudo apt install portaudio19-dev python3-pyaudio
```

### Python Environment
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
